# Flight Fare Prediction

## Objective
Predict the fare of a flight by analyzing the past data and using Machine Learning.

## Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Flask
- HTML
- CSS

## Machine Learning Algorithm
Random Forest Regressor

## Model Performance
R² Score : 0.8143
MAE : 1171
RMSE : 1967

## Features Used
- Airline
- Source
- Destination
- Journey Date
- Departure Time
- Arrival Time
- Duration
- Total Stops

## How to Run

pip install -r requirements.txt

python app.py
